Floating Messages — Usage Instructions

The 'Messages' folder contains the phrase files that the goose can display on screen.
Each file must have the .txt extension and contain one phrase per line.

Rules:

1. Each line represents a phrase the goose can say.
2. Empty files or files with only spaces will be ignored.
3. Files inside the "Ignored" subfolder are also ignored.

Tips:

- You can organize phrases by theme by creating multiple .txt files.
- Avoid very long phrases for better visibility.

Have fun!

Floating Messages — Instruções de Uso

A pasta 'Messages' contém os arquivos de frases que o ganso pode exibir na tela.
Cada arquivo deve ter a extensão .txt e conter uma frase por linha.

Regras:

1. Cada linha representa uma frase que o ganso pode dizer.
2. Arquivos vazios ou com apenas espaços serão ignorados.
3. Arquivos dentro da subpasta "Ignored" também são ignorados.

Dicas:

- Você pode organizar as frases por tema criando vários arquivos .txt.
- Evite frases muito longas para melhor visibilidade.

Divirta-se!
